01.   what are the inline and block elements in html and difference between them? Name a few inline element and block elements

Block-level elements are those elements that typically occupy the entire width of their parent container and create a new line after the element.
Examples of block-level elements : div, p, ul, li, header, footer, nav, section, article, and so on.

Inline elements, on the other hand, are those elements that only take up the necessary width to display their content and do not create a new line after the element.
Examples of inline elements : a, span, img, strong, em, label, input, and so on.

Diffrent between block and inline element

Block Elements Inline Elements
Block elements always start from a new line. Inline elements never start from a new line.
Block elements cover space from left to right as far as it can go. Inline elements only cover the space as bounded by the tags in the HTML element.
Block elements have top and bottom margins. Inline elements don't have a top and bottom margin.
Examples of block elements - <p>,<div>,<hr> . Examples of inline elements - <span>,<br>

02.   How to work with images in html and explain in details <img> tag important attributes.

Adding images on a webpage: The <img> tag is used to add or embed the images to a webpage/website. The “img” tag is an empty tag, which means it can contain only a list of attributes and it has no closing tag. The addition of the images improves the quality along with enhancing the design structure, appearance of the webpage. Nowadays, a website does not directly add images to a web page, as the images are linked to web pages by using the <img> tag which holds space for the image. 

  • src: This is a required attribute that specifies the source of the image. It can be a file path or a URL to an image.

  • alt: This attribute provides an alternative text description of the image. It is used when the image cannot be displayed or for users who are using screen readers.

  • title: This attribute provides a tooltip that is displayed when the user hovers over the image.

  • width and height: These attributes specify the width and height of the image in pixels. They can be used to resize the image on the page.

  • loading: This attribute is used to control how the image is loaded. It can have three values: "lazy" to defer loading until the image enters the viewport, "eager" to load the image immediately, or "auto" to use the browser's default behavior.

  • decoding: This attribute is used to control how the image is decoded. It can have two values: "sync" to decode the image as soon as it is loaded, or "async" to defer decoding until the image is displayed.

  • sizes: This attribute is used with the srcset attribute to specify the sizes of the image on different devices. It can be a list of media queries or a single value indicating the default size.

03.   How to create lists in HTML?

In HTML, there are three types of lists: ordered lists, unordered lists, and definition lists. Here are the steps to create each type of list:

  1. Unordered lists: An unordered list is a list of items that are marked with bullets. To create an unordered list in HTML, use the <ul> tag and add <li> tags for each list item. For example:

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ul>
  2. Ordered lists: An ordered list is a list of items that are numbered. To create an ordered list in HTML, use the <ol> tag and add <li> tags for each list item. For example:

    <ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ol>
  3. Definition lists: A definition list is a list of terms and their definitions. To create a definition list in HTML, use the <dl> tag and add <dt> tags for each term and <dd> tags for each definition. For example:

    <dl>
    <dt>Term 1</dt>
    <dd>Definition 1</dd>
    <dt>Term 2</dt>
    <dd>Definition 2</dd>
    <dt>Term 3</dt>
    <dd>Definition 3</dd>
    </dl>
