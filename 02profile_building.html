1. what is the difference between UI and UX? Explain in your own words

Ans = UI is the visual and interactive part of a digital product, which includes the layout, typography, color scheme, icons, buttons, and other visual elements that users see and interact with. UI designers are responsible for creating an attractive and intuitive interface that helps users navigate and use the product easily. A good UI design should be aesthetically pleasing, consistent, and responsive to user actions.

On the other hand, UX is the overall experience that a user has while using a product, which includes the UI design, as well as the ease of use, functionality, performance, accessibility, and other factors that impact the user's satisfaction with the product. UX designers focus on understanding user needs and behavior, testing and iterating design solutions, and creating a seamless and enjoyable user experience.

2. Who is a frontend developer? Explain the roles and responsibilities in your own words.

Ans = A Frontend Developer is a web developer who specializes in designing and developing the client-side or user-facing portion of a web application or website. They are responsible for creating the look, feel, and functionality of the web pages that users see and interact with.

The roles and responsibilities of a frontend developer

  • Developing the user interface
  • Creating web pages
  • Optimizing web pages for speed and performance
  • Writing clean and maintainable code

5. What is dynmaic site? Explain in your own words.

Ans = A dynamic site is a type of website that generates content on the fly using server-side scripts and databases. Dynamic sites use programming languages such as PHP, Ruby, Python, or JavaScript to generate HTML, CSS, and JavaScript files on the server-side. These files are then sent to the user's browser, where they are displayed as web pages.
